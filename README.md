
# SEMO_io

SEMO_io is header only library for mesh i/o.

## Requirements

- C++20 compiler
- VTK 9.2 version
- tecio

## Available mesh file types

- Ansys fluent
    - ascii/binary read/write
- OpenFoam
    - ascii read/write
- Tecplot
    - ascii/binary write
- VTU
    - ascii/binary read/write
- STL
    - ascii/binary read/write
- ply
    - ascii read/write
- obj
    - ascii read/write
- off
    - ascii read/write
