
# SEMO_io

SEMO_io is header only library for mesh i/o.

## Requirements

- C++20 compiler
- VTK 9.2 version
- tecio

## Available mesh file types

- Ansys fluent ¡æ ascii/binary read/write
- OpenFoam ¡æ ascii read/write
- Tecplot ¡æ ascii/binary write
- VTU ¡æ ascii/binary read/write
- STL ¡æ ascii/binary read/write
- ply ¡æ ascii read/write
- obj ¡æ ascii read/write
- off ¡æ ascii read/write
